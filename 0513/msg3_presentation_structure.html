簡報架構規劃建議

針對 Msg3 攔截攻擊 (Opportunistic Attack) 與防禦機制的展示架構
投影片 1

Msg3 攔截與破壞攻擊：實作與機制 (Msg3 Hijacking & Corruption Attack)

攻擊概念 (Concept)

  • 攻擊者 不主動發送 PRACH (Msg1) 以節省發射能量並隱蔽行蹤。
  • 寄生攔截 (Parasitic Interception)： 攻擊者持續監聽網路中的 RAR (Msg2)，只要收到任何 RAR，無論其 RAPID 是否匹配，皆強制接收並占用該 Uplink Grant (程式碼修改：rarh->RAPID >= 0)。

破壞手段 (Execution)

  • 在 PHY 層攔截 Msg3 Payload，將原本的資料與 CRC 替換為 隨機亂碼 (rand() % 256)。
  • 利用攔截到的 Uplink Grant 發送損壞的 Msg3，導致 gNB 發生 CRC Error 且無法解碼，進而引發無效的 HARQ 重傳。
  • MAC 層無限迴圈： 移除 Msg3 傳輸狀態的更新，並大幅增加 preambleTransMax (例如調整為 3000 以上)，讓攻擊 UE 永遠處於等待與攔截狀態。

視覺化建議

畫一個簡單的循序圖 (Sequence Diagram)，顯示合法 UE 發送 Msg1，gNB 回覆 Msg2 (RAR)，但 攻擊者搶先用這個 RAR 的 Grant 發送了亂碼 Msg3，導致合法 UE 的 Msg3 發生碰撞或 gNB 資源被浪費。

投影片 2

基於 O-RAN 架構之 Msg3 異常偵測與緩解 (Msg3 Anomaly Detection & Mitigation)

偵測邏輯 (Detection Logic)

  • 此攻擊會導致 gNB 成功發送 RAR (ra_initiated 增加)，但隨後的 Msg3 永遠無法成功解碼 (CRC Failure 飆高)。
  • rApp 透過 O1/E2 介面監控 KPI：若發現 RAR 發送數量正常，但 RRC Setup Request (Msg3) 解碼成功率趨近於 0 (或是 HARQ 失敗率極高)，即觸發 Msg3 攻擊警報。

緩解機制 (Mitigation Action)

  • rApp 下達指令重啟/重配 gNB 的 RACH 參數。
  • 動態調整 PRACH Occasion，或是利用 MAC 層排程器動態黑名單過濾異常的 Uplink 行為，恢復合法 UE 的連線能力。
投影片 3

Msg3 攻擊測試結果驗證 (Validation of Msg3 Attack)

Attacker (UE) 端數據展示

  • 放上自定義的 LOG 截圖，標註關鍵字：
    • Payload modified (證明成功攔截並進入修改邏輯)
    • Modified Payload (payload_AB): 隨機 Hex Data (證明 Payload 被成功置換)

Victim (gNB) 端數據展示

  • 展示 gNB 端的 Log 顯示 PUSCH CRC Failure 或是 Msg3 解碼失敗 / HARQ Timeout。
  • 影響數據圖表： 展示在攻擊開啟時，合法 UE 的連線成功率 (Connection Success Rate) 瞬間掉到 0%，或是 gNB 的 PUSCH 資源利用率被無效資料佔滿。
投影片 4

系統恢復與緩解成效 (Mitigation Effectiveness & System Recovery)

觸發緩解的 Log

  • 展示 rApp (或 SMO) 偵測到 Msg3 異常並發出警報 (prachAttackDetected / Msg3 CRC anomaly) 的紀錄。

恢復正常連線的圖表/數據

  • 以時間軸圖表 (Time-series plot) 呈現三個階段：
    1. Phase 1 (正常)：連線成功率 100%。
    2. Phase 2 (攻擊開始)：Msg3 CRC Error 飆升，連線成功率降至 0%。
    3. Phase 3 (緩解觸發)：系統套用防禦機制，惡意流量被阻擋或規避，合法 UE 連線成功率恢復。

💡 報告排版小建議

您可以把程式碼 Diff 中最精華的部分擷取出來，例如 harq_process->payload_AB[i] = rand() % 256; 以及 if (rarh->RAPID >= 0)，用紅框圈起來放在「投影片 1」旁邊。
這樣指導教授或台下聽眾一看就能馬上理解這個攻擊是如何在 OAI 源碼中以極低的成本（改幾行 Code）造成極大的破壞，非常有說服力。
